# Python Flask MySQL Projects

This repository contains backend web development projects built using Python, Flask, and MySQL.

The projects were created while learning server-side development, database integration, and CRUD application design.

--------------------------------------------------

PROJECTS INCLUDED

First Flask MySQL
An introductory project demonstrating how Flask connects to a MySQL database.

CRUD Applications
Several projects demonstrating Create, Read, Update, and Delete (CRUD) operations.

Dojos and Ninjas
A relational database project demonstrating relationships between models.

Login and Registration
A user authentication system that allows users to register and log in.

Pies
A simple web application demonstrating form handling and database interaction.

Recipe / Recipes
Applications that allow users to create, view, and manage recipes.

Session
A project demonstrating session management in Flask.

Users
An example project demonstrating user data management with Flask and MySQL.

Sasquatch
A small project demonstrating database interaction and web application structure.

Scuba Divers
A database-driven project demonstrating relationships and queries.

GOAL
